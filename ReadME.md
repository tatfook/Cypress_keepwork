# 本项目主要是采用Cypress框架来验证Keepwork 常用的功能

## Getting Started
> 本地已经有node环境及git工具
- 拷贝项目到本地
  ```
  git clone  Url(项目地址)
  ```
- 安装相关依赖(推荐以yarn方式安装)
  + 以npm为例
    ```
        npm install
    ```
  + 以yarn为例
    ```
       yarn add
    ```
- 运行命令
  + window环境  
    执行start.bat或者命令行输入```npx run```
  + linux环境  
    执行start.sh或者命令行输入```yarn run```
  + docker环境  
    待补充
